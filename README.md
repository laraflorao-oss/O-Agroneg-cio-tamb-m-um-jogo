# Tema do meu projeto

*O que é o Projeto

Este site é um projeto feito para o Concurso Agrinho com o tema "O Agronegócio também é um Jogo". A ideia principal é mostrar que o agronegócio não joga sozinho e não é o único culpado pela poluição do planeta. O projeto defende que o cuidado com o meio ambiente é uma responsabilidade de todos, pois o nosso estilo de vida na cidade e o uso de novas tecnologias também gastam muitos recursos naturais e poluem o mundo, mesmo que ninguém fale disso.

¨¨Como foi Desenvolvido

O projeto foi desenvolvido pensando em criar um site simples, direto e fácil de navegar. O conteúdo foi organizado para fazer o visitante refletir, começando pela crítica sobre como a sociedade culpa o agricultor para esconder a poluição das cidades, mostrando dados reais sobre o consumo de água e terminando com um alerta sobre a falta de apoio para a tecnologia no campo.

"Tecnologias e Linguagens Utilizadas

Para construir o site, foram usadas as ferramentas básicas da programação web:

HTML5: Para criar a estrutura do site e organizar os textos.

CSS3: Para cuidar do visual, das cores e fazer o site funcionar bem no computador e no celular.

JavaScript: Para deixar o jogo educativo mais dinâmico.

_O que o Projeto Contempla

O site traz três pontos principais:

O Jogo de Interesses: Explica que o agronegócio segue regras do mercado mundial e que as pessoas acham mais fácil julgar o agricultor do que diminuir o lixo e a poluição que geram nas cidades.

O Comparativo da Água (IA vs. Agricultura): Mostra um exemplo real de como a tecnologia gasta água. Enquanto 1 hectare de alface gasta 1,8 milhão de litros de água em 30 dias para produzir alimento para 5.000 pessoas, uma Inteligência Artificial gasta 75.000 litros em apenas uma hora para resfriar seus servidores. Isso significa que a água que a IA gasta em 1 hora daria para irrigar a plantação de alface por mais de um dia inteiro.

A Falta de Incentivo: Mostra que existem tecnologias como drones e irrigação inteligente que ajudariam o campo a poluir ainda menos. Porém, o governo e a sociedade não investem nisso porque, nesse jogo de interesses econômicos e políticos, eles acham que estariam perdendo dinheiro ou poder.
